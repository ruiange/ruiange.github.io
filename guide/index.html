        # 指南

        # 介绍

        Hi，欢迎使用“酱茄小程序开源版 – 人人咖啡师”。

        酱茄小程序开源版基于 WordPress 开源程序和 WordPress REST API 开发，实现 WordPress 网站数据与小程序数据同步共享，通过简单的配置就能搭建自己的小程序。

        # 快速上手

        如果不了解如何安装 wordpress，可上网搜索，本文档不会讲解如何安装 wordpress

        # 准备工作

        1. 环境选择：

          推荐使用 宝塔面板

          类型 建议使用
          服务器操作系统 CentOS 7.7
          WEB 服务器 nginx
          PHP 版本 推荐使用 7.1，7.1 以下会出现兼容性错误
          数据库 MySQL5.6

        2. 小程序平台配置

          • 首先需要一个小程序帐号，小程序帐号注册流程

          • 注册完成后 打开小程序后台 点击左侧的 "</>开发" 菜单,然后 "开发设置" >服务器域名

          • 添加你的网站地址

        1. 域名证书 小程序请求域名须为 HTTPS 协议 阿里云 腾讯云这些平台上可以申请一年的免费 SSL 证书。 如果你是宝塔面板，可一键申请域名并完成配置。

        # 下载代码

        github 下载 gitee 下载

        点击绿色的 Clone or download 按钮下载 ZIP 文件，就会得到一个压缩包。

        打开压缩包 api 文件夹下的 jiangqie-api 文件夹就是我们需要安装到 wordpress 的插件，把文件夹用 ftp 上传到网站的 wp-content/plugins 下。 然后在网站后台安装此插件。

        # 插件配置

        启用插件后，打开酱茄设置》基础设置

        填入 APPID 和 AppSecret

        # 如何获取 APPID 和 AppSecret？

        登入小程序公众平台后台，点击左侧菜单 开发 》 然后点击开发设置

        获取后 到Wordpress后台酱切设置里填入。

        此时 你已经完成一半的配置了。

        # 小程序

        下载微信开发工具

        安装完成后选择“导入项目”

        导入我们从github上下载的文件
        填写APPID

        # 修改小程序域名

        打开 在utils下的config.js文件,把域名修改成你网站的域名,不需要带hhtp或者https前缀

        # 上线

        域名修改后，就已经小程序的所有配置。 点击开发工具右上角的上传

        等待上传完成后，打开小程序官方后台 点击左侧 管理》版本管理，然后点击提交审核,审核完成后,大概等待一至两天,微信就会完成审核,此时就可以把小程序发布上线了。
