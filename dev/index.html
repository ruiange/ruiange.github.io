      # 个人资质小程序如何打开网站

      小程序增加了打开公众号文章功能,我们可通过插入公众号文章,在文章底部的“阅读原文”插入网站链接。

      # 个人资质如何插入公众号文章

      酱茄小程序 有个“活动区域图”模块 就是左边一个图 右边两个图的模块，我们可以再次基础上添加跳转公众号文章的方法。 在openLink方法 对传入的url做判读 如果为微信链接 则直接跳转web-view（web-viwe组件请查阅官方文档如何使用）

          openLink: function (link) {
        let url = link
        if (link.startsWith('https://mp.weixin.qq.com')) {
            wx.navigateTo({
                url: `../webView/index?url=${url}`,
            })
        } else if (link.startsWith('/pages')) {
            wx.navigateTo({
                url: link,
            })
        } else {
            wx.navigateToMiniProgram({
                appId: link,
                fail: res => {
                    wx.showToast({
                        title: '无效链接',
                    })
                }
            })
        }
    }

      # 分享海报功能

      实际效果图（小程序还未发布上线）

      小程序海报生成基于canvas，不依赖后端 3S之内就可生成海报分享图。

      本项目基于mp_canvas_drawer Canvas绘图组件，引入组件后，传入少量的参数就可生成一张海报分享图。
