# 云端切换请求域名

可在小程序上线后,切换小程序绑定的网站域名。（缺点:会增加请求时间）

  1. 打开微信开发着工具，点击云开发。初始化云开发环境。

  2. 新键项目 ，选择 小程序-云开发

  3. 右键点击 cloudfunctions 文件夹 选择新键 NodeJS 云函数"switchDomain"

  4. 删除原有代码 复制以下代码 保存。

    // 云函数入口文件
const cloud = require('wx-server-sdk')

cloud.init()

// 云函数入口函数
exports.main = async (event, context) => {

  return {
    JQ_DOMAIN:'填写你的域名你的域名' //此处填写你的域名
  }
}
